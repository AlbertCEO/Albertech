Our Courses/Stacks

  • HTML, CSS & JavaScript

    Students will learn the foundational structure of a website by implementing HTML, CSS and Javascript.

  • CSS Frameworks

    In this course, students will learn current CSS frameworks like Bootstrap, Tailwindcss, Vanilla and more libraries used to develop Responsive Mobile-Friendly Applications

  • JavaScript Frameworks

    Students will be exposed to JavaScript frameworks like ReactJs, React Native, NextJs Angular, Vue and more. Attention will be paid to NextJs as it's one of th leading framework now.

  • Backend Programming

    Students will be exposed to Full Stack Programming and will be taught NodeJs and MongoDb Database. Student will learn advance techniques in fetching data from the backend to the frontend.

  • React Native

    In this course, students will be taught how to develop cross-platform mobile applilcation with the use of React Native and other libraries that will be introduced in this course.

  • PHP Programming

    PHP is one of the popular scripting languages used to develop robost and advanced database driven websites. Students will learn how to develop content-based management application with the help of PHP.
